# odin-recipes
## Introduction
The **odin-recipes** is a simple four paged website built purely with HTML.
It has `index.html` as the homepage contaning 3 links to recipe pages namely:
1. `pilau.html` - For pilau recipe
2. `ugali.html` - For ugali recipe
3. `sukuma.html` - Sukuma wiki recipe page

All the 3 recipe pages have a *home* link which takes user back to `index.html`(homepage) after viewing the recipe.
